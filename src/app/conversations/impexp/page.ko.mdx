## 가져오기/내보내기

이 섹션에서는 erxes XOS에서 데이터를 가져오고 내보내는 방법을 배웁니다.

<Note>
가져오기 및 내보내기 기능은 Excel 스프레드시트를 사용하여 erxes에서 데이터를 가져오고 내보내도록 설계되었습니다. Cards 플러그인 및 Contacts 플러그인에서 모든 것을 가져올 수 있습니다.
</Note>

### 가져오기
1. 설정 메뉴에서 가져오기 기능으로 이동합니다.
![](https://imagedelivery.net/5m26Aj-CutMXPPNacMs_yQ/9ca2dd96-1add-48fc-e89c-b0f280b15a00/public)

2. 가져오려는 유형을 선택합니다. 사용할 수 있는 유형은 다음과 같습니다:
- 거래 판매 필드
- 작업 활동 필드
- 티켓 피드백 필드
- 고객 사용자 데이터
- 리드 통합을 생성한 사람에 대한 데이터
- 회사의 고객 및 기타 제휴 회사
- 제품 가져오기

![](https://imagedelivery.net/5m26Aj-CutMXPPNacMs_yQ/604dfbe6-ff43-4bbf-cc8e-1296c60fc200/public)

3. 사용자 가져오기를 예로 들어보겠습니다. 필터에서 콘텐츠 유형별로 고객을 선택합니다.
![](https://imagedelivery.net/5m26Aj-CutMXPPNacMs_yQ/c4db76a3-1ece-47be-471c-b55c807a6300/public)

4. 다시 고객을 선택하고 다음을 클릭합니다.
![](https://imagedelivery.net/5m26Aj-CutMXPPNacMs_yQ/02337828-dffb-4901-c986-8740fb701800/public)

5. 가져올 템플릿을 지정하지 않은 경우, 템플릿 다운로드로 이동합니다.
![](https://imagedelivery.net/5m26Aj-CutMXPPNacMs_yQ/bf721b6f-42d4-46c9-084b-fa2ff5d26000/public)

6. 가져올 헤더를 확인합니다.
![](https://imagedelivery.net/5m26Aj-CutMXPPNacMs_yQ/2880fdb6-6d40-4710-724c-72e2ee020300/public)

7. Excel을 다운로드하고 준비한 후 데이터를 입력합니다. CSV 확장자만 사용 가능합니다.
![](https://imagedelivery.net/5m26Aj-CutMXPPNacMs_yQ/6fb38341-1ff5-4f83-c014-e7b9b5b9d800/public)

8. 다음 단계에서는 Excel 헤더를 테이블의 데이터 헤더와 매칭합니다.
![](https://imagedelivery.net/5m26Aj-CutMXPPNacMs_yQ/c41d9a4d-4ad6-4b44-a821-2aeef5a9dc00/public)

9. 다음을 클릭하고, 이름과 고지를 확인한 후 가져오기를 클릭합니다.
![](https://imagedelivery.net/5m26Aj-CutMXPPNacMs_yQ/6ca3c177-15ec-46c6-e454-1b20481dd800/public)

10. 여기에서 가져오기 진행 상태를 확인할 수 있습니다.
![](https://imagedelivery.net/5m26Aj-CutMXPPNacMs_yQ/acb6e10f-bf56-424e-094a-56a8ac7ca400/public)

11. 데이터는 기록, 업데이트 및 오류 지표로 나뉩니다.
![](https://imagedelivery.net/5m26Aj-CutMXPPNacMs_yQ/276734a6-07f3-4699-1276-1908a9f0e800/public)

12. 생성된 가져오기 작업 섹션 - 연락처 다운로드: 고객 파일 - 설치 파일 - 연락처 다운로드: 고객 오류 - 누락된 파일의 오류 값 설치 - 연락처 삭제: 고객 - 가져오기 삭제
![](https://imagedelivery.net/5m26Aj-CutMXPPNacMs_yQ/4c4f0e1c-98c4-4c0d-1277-70d6a47b4300/public)

### 내보내기
<Tip>
이 섹션에서는 erxes XOS에서 데이터를 내보내는 방법을 배웁니다. erxes 내보내기는 두 가지 구성 요소로 이루어져 있습니다:

콘텐츠: 내보낼 데이터
필터: 내보내기에 사용할 필터
</Tip>

1. erxes 패널의 왼쪽 하단에 위치한 설정에서 가져오기 및 내보내기 섹션으로 이동합니다.
![](https://imagedelivery.net/5m26Aj-CutMXPPNacMs_yQ/1c59bdea-4b9c-43e4-10ff-c62752018600/public)

2. 회사에 대한 데이터를 내보내 봅시다. 콘텐츠 유형별 필터에서 회사를 선택합니다.
![](https://imagedelivery.net/5m26Aj-CutMXPPNacMs_yQ/a71cceaf-7da3-4d81-a9f8-2faabb8fac00/public)

3. "내보내기" 탭을 클릭합니다.
![](https://imagedelivery.net/5m26Aj-CutMXPPNacMs_yQ/bab5a05d-66ee-4de6-33c1-4563196e6c00/public)

4. 콘텐츠는 erxes XOS에서 내보낼 Excel 시트의 헤더입니다. 내보낼 정보를 확인합니다. 내보낼 모든 정보를 선택한 후 "다음"을 클릭합니다.
![](https://imagedelivery.net/5m26Aj-CutMXPPNacMs_yQ/4b35f780-9e07-41bc-0836-bb6c9e462f00/public)

5. 다음 페이지의 필터 섹션에서 가져올 필터를 입력합니다.
![](https://imagedelivery.net/5m26Aj-CutMXPPNacMs_yQ/b8cfcbab-d420-4f21-c54b-0a957be3b800/public)

6. 일반 및 사용자 정의 속성 섹션에 필터 값을 입력합니다.
![](https://imagedelivery.net/5m26Aj-CutMXPPNacMs_yQ/65e59f7f-1a30-45b5-b8b9-61748df3ab00/public)

7. 이 경우 필터에 포함된 모든 이름을 검색하기 위해 값을 입력합니다.
![](https://imagedelivery.net/5m26Aj-CutMXPPNacMs_yQ/3fb9d3d1-9d93-44ba-7262-b8429a8ff000/public)

8. 필터링한 후, 필터 적용을 클릭합니다.
![](https://imagedelivery.net/5m26Aj-CutMXPPNacMs_yQ/df2686a1-d6cc-40d4-4413-d71b36095800/public)

9. 다시 적용을 클릭합니다. 내보내기를 위해 필터 적용을 두 번 클릭합니다. 하나는 필터용이고, 다른 하나는 필터 적용 확인용입니다.
![](https://imagedelivery.net/5m26Aj-CutMXPPNacMs_yQ/5c752333-81bd-469c-31e5-a5e003df1c00/public)

10. 필터에서 찾은 정보를 확인할 수 있습니다.
![](https://imagedelivery.net/5m26Aj-CutMXPPNacMs_yQ/360852d1-8857-47c8-a2af-e82a49848800/public)

11. 마지막으로 "내보내기"를 클릭하면 선택한 파일이 컴퓨터로 내보내집니다.
![](https://imagedelivery.net/5m26Aj-CutMXPPNacMs_yQ/6da8bfa6-ffb4-41cd-3b8a-2c1ff68a5700/public)
