# **사용자 관리**

역할, 권한 및 인증 메커니즘을 통해 시스템에 대한 사용자 접근을 효율적으로 제어하고 조직하여 각 사용자에게 안전하고 맞춤화된 경험을 보장합니다.
![](https://office.erxes.io/gateway/read-file?key=office-erxes-io/0.8217520008815253Screenshot2023-12-03124606.png)

사용자 인증:
- 안전한 로그인 메커니즘.
- 암호 정책 및 암호화.
- 다중 인증.

사용자 역할 및 권한:
- 사용자 역할 정의 (관리자, 매니저, 직원 등).
- 각 역할에 대한 접근 수준 지정.
- 역할 할당 및 관리.
