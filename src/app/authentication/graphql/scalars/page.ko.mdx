# Scalars

## Boolean
`Boolean` 스칼라 타입은 `true` 또는 `false`를 나타냅니다.

## Date
날짜 커스텀 스칼라 타입

## Float
`Float` 스칼라 타입은 [IEEE 754](https://en.wikipedia.org/wiki/IEEE_754)에 명시된 대로 서명된 배정밀도 소수 값을 나타냅니다.

## Int
`Int` 스칼라 타입은 부호가 있는 정수를 나타냅니다. Int는 -(2^31)에서 2^31 - 1 사이의 값을 나타낼 수 있습니다.

## JSON
`JSON` 스칼라 타입은 [ECMA-404](https://ecma-international.org/publications/files/ECMA-ST/ECMA-404.pdf)에 명시된 대로 JSON 값을 나타냅니다.

## String
`String` 스칼라 타입은 UTF-8 문자 시퀀스로 표현된 텍스트 데이터를 나타냅니다. String 타입은 주로 GraphQL에서 자유형 인간 가독 텍스트를 나타내는 데 사용됩니다.
