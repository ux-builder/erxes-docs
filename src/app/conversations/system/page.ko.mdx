## 시스템 설정

시스템 설정은 이메일 설정, 언어, 첨부 파일 등 erxes XOS의 일반 설정을 구성하는 곳입니다.

<Note>
erxes에 가입하면 평균적인 회사에서 필요한 기본 설정이 모두 미리 구성되어 있습니다. 비즈니스 요구 사항을 충족하는지 확인하세요. 추가 설정이 필요하면 고객 서비스 팀에 문의하세요.
</Note>

시스템 설정에는 다음 필드가 있습니다:

- 일반 설정
- 파일 업로드
- 구글 클라우드 스토리지
- AWS S3
- AWS SES
- 구글
- 일반 메일 설정
- 사용자 정의 메일 서비스
- 데이터 보존
- 상수
- 연결 서비스
- 메시지프로

**1. 관리자 화면 왼쪽 하단의 설정 섹션에서 시스템 설정으로 이동하세요.**

**2. 일반 설정**
- **언어** - erxes XOS에서 사용할 언어를 선택하세요.
- **통화** - 제품 및 서비스, 판매 파이프라인 플러그인에 사용할 통화를 선택하세요.
- **측정 단위** - 판매 파이프라인에서 사용할 판매 측정 단위를 선택하세요.
![](https://imagedelivery.net/5m26Aj-CutMXPPNacMs_yQ/5f9dd854-011d-4a59-29c0-42747b622c00/public)
**3. 구글**
구글 클라우드 플랫폼(GCP)은 구글이 내부적으로 사용하는 인프라를 기반으로 하는 클라우드 컴퓨팅 서비스 모음입니다. Firebase, Gmail, Pubsub과 같은 최종 사용자 제품을 위해 내부적으로 사용하는 인프라에서 실행됩니다. 관리 도구 세트와 함께 컴퓨팅, 데이터 저장소, 데이터 분석, 머신 러닝 등을 포함하는 모듈형 클라우드 서비스를 제공합니다.
다음 단계는 Erxes 앱에서 구글 클라우드 플랫폼 서비스를 사용하도록 구글 클라우드 프로젝트를 설정하는 방법을 설명합니다.

**5. 구글 클라우드 스토리지**
클라우드 스토리지는 전 세계적으로 내구성이 높은 객체 스토리지를 제공하며 엑사바이트 데이터까지 확장 가능합니다. 모든 스토리지 클래스에서 데이터를 즉시 액세스할 수 있으며, 단일 통합 API로 애플리케이션에 스토리지를 통합하고 가격 및 성능을 쉽게 최적화할 수 있습니다.
