# 응답 템플릿
응답 템플릿 기능은 고객, 잠재 고객 및 직원과의 일상적인 대화를 위해 응답 템플릿을 생성하는 데 도움을 줍니다.

## 응답 템플릿으로 할 수 있는 일
고객의 문의에 신속하게 응답하는 것은 고객 만족도와 유지율을 높이는 데 매우 중요합니다. erxes 응답 템플릿을 사용하여 고객의 문의에 응답하고 업데이트하면 더 행복한 삶을 누릴 수 있습니다.

- FAQ에 응답 템플릿 필요할 때 응답 템플릿을 FAQ에 사용하면 많은 시간과 비용을 절약할 수 있습니다.
- 응답 템플릿은 인간 오류를 방지하며 응답 템플릿은 작업 단계를 자동화할 때 판매 파이프라인/작업 관리/티켓 플러그인과 함께 사용됩니다.
