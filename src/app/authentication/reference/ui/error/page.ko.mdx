# 에러 메시지
오류가 발생했음을 알리기 위해 에러 메시지를 제공합니다.

## **예시**

```tsx
<ErrorMsg>이것은 에러입니다</ErrorMsg>
```

## **API**

```tsx
import ErrorMsg from "erxes-ui/lib/components/ErrorMsg";
```
* required prop

| 이름 | 유형 | 기본값 | 설명 |
| ---| --- | ---| --- |
| children*	 | string |  | 에러 메시지를 선언합니다 |
