# 수정 가능한 목록
## **옵션**
`options` 속성을 사용하여 목록을 삽입합니다.

```tsx
<>
	<ModifiableList options={["Name","Age"]} />
</>
```

## **추가 버튼 라벨**

"취소" 버튼을 클릭하면 세 번째 버튼이 나타납니다. `addButtonLabel` 속성을 사용하여 세 번째 버튼의 라벨을 변경할 수 있습니다.

```tsx

<>
	<ModifiableList options={["Name","Age"]} addButtonLabel="Custom button label" />
</>

```

## **API**

```tsx
import ModifiableList from "erxes-ui/lib/components/ModifiableList";
```
* 필수 속성

| 이름 | 유형 | 기본값 | 설명 |
| ---| --- | ---| --- |
| options* | string[] |  | 시작 시 옵션을 표시합니다 |
| addButtonLabel | string | "Add an option"	 | 세 번째 버튼 라벨을 변경합니다 |
| onChangeOption | function| | 삭제 아이콘을 클릭할 때 실행되는 함수를 정의합니다 |
