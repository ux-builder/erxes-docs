# 테이블

콜론을 사용하여 열을 정렬할 수 있습니다.

| 테이블 | 는 | 멋집니다 |
| ---- | ---- | ----------- |
| col 3은 | 오른쪽 정렬됨 | \$1600 |
| col 2는 | 가운데 정렬됨 | $12 |
| 얼룩무늬 | 멋짐 | $1 |
