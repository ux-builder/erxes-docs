# Name card
## **Username**
`username` 속성을 사용하여 사용자 이름을 표시합니다.

```tsx
<>
	<NameCard  user={{username:"james"}} />
</>
```

## **Full name**
`fullName` 속성을 사용하여 전체 이름을 표시합니다.

```tsx
<>
	<NameCard  user={{details:{fullName:"James Smith"}}} />
</>
```

## **Avatar size**
`avatarSize` 속성을 사용하여 아바타 크기를 변경할 수 있습니다.

```tsx
<>
	<NameCard  avatarSize={50} user={{details:{fullName:"James Smith"}}} />
</>
```

## **User E-mail**
`email` 속성을 사용하여 사용자 이메일을 표시합니다.

```tsx
<>
	<NameCard  user={{details:{fullName:"James Smith"},email:"jamessmith@gmail.com"}} />
</>
```

## **Single Line**
`singleLine` 속성을 사용하여 두 번째 줄을 보이지 않게 할 수 있습니다.

```tsx
<>
	<NameCard  singleLine={true} user={{details:{fullName:"James Smith"},email:"jamessmith@gmail.com"}} />
</>
```

## **Second line**
`secondLine` 속성을 사용하여 두 번째 줄을 표시합니다.

```tsx
<>
	<NameCard  secondLine="Intern" user={{details:{fullName:"James Smith"}}} />
</>
```

## **API**

```tsx
import Namecard from "erxes-ui/lib/components/namecard/Namecard";
```
* 필수 속성

| 이름 | 유형 | 기본값 | 설명 |
| ---| --- | ---| --- |
| username*	 | string |  | 사용자 이름을 정의합니다 |
| fullName | string |  | 사용자 전체 이름을 정의합니다. "details"와 "username" 둘 다 있는 경우, "details"만 표시됩니다 |
| email | string| | 사용자 이메일을 정의합니다 |
| singleLine | string |  | 두 번째 줄을 보이지 않게 합니다 |
| secondLine | string |  | 사용자 이름이나 전체 이름 아래에 표시될 줄을 정의합니다. 두 번째 줄에 아무 내용이나 쓸 수 있습니다 |
| avatarSize | number | 40  | 이름 카드의 아바타 크기를 설정합니다 |
