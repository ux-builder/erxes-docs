# 스킬 소개
스킬은 erxes 메신저 플러그인의 기능 확장으로, 수신된 대화를 적절한 팀 및 고객에게 전달하는 데 도움을 줍니다. 예를 들어:
- 언어 스킬
- 관련 팀
## 스킬 설정 방법
1. 관리자 패널의 왼쪽 하단에 있는 설정 섹션에서 스킬을 선택합니다.
![](https://imagedelivery.net/5m26Aj-CutMXPPNacMs_yQ/c92c707f-5997-47a1-808c-cc264d7f1a00/public)
2. 필요한 다양한 스킬 유형에 따라 스킬 유형을 생성합니다.
![](https://imagedelivery.net/5m26Aj-CutMXPPNacMs_yQ/c9065325-30e3-4399-2e9c-4d9b046d9000/public)
3. 관련 팀과 함께 새로운 스킬 유형을 생성합니다.
![](https://imagedelivery.net/5m26Aj-CutMXPPNacMs_yQ/3b0d461a-a42e-457e-03af-f1363aaadb00/public)
4. "새로운 스킬"을 눌러 스킬을 생성합니다.
![](https://imagedelivery.net/5m26Aj-CutMXPPNacMs_yQ/f546ea2d-370b-45c7-b7c2-ba39078f4600/public)
5. 이 스킬에 속하는 팀원을 추가합니다. 고객이 세일즈 팀과 소통하기를 선택하면, 세일즈 팀에 속한 팀원들이 소통을 받게 됩니다.
![](https://imagedelivery.net/5m26Aj-CutMXPPNacMs_yQ/c787d2ff-f6c1-4d3b-402e-e8f056f48d00/public)
6. 스킬 유형은 세일즈, 마케팅, 지원 등으로 팀 단위로 나뉩니다.
![](https://imagedelivery.net/5m26Aj-CutMXPPNacMs_yQ/15783964-eb67-4a03-80e6-de6297519000/public)
7. 이후, 스킬이 웹 메신저에 표시됩니다. 이를 위해 통합 플러그인으로 이동합니다.
![](https://imagedelivery.net/5m26Aj-CutMXPPNacMs_yQ/bf406af3-3398-4eb6-0205-eb5e29eee200/public)
8. 메신저를 선택하고 스킬 입력 메신저를 편집합니다.
![](https://imagedelivery.net/5m26Aj-CutMXPPNacMs_yQ/9d3efff5-30c8-4b7b-221f-bc9b59d47300/public)
9. 인트로 필드로 이동합니다.
![](https://imagedelivery.net/5m26Aj-CutMXPPNacMs_yQ/91eb809a-9fb1-4f9a-b2cf-54dd2d3c8f00/public)
10. 아래에서 "메신저에 스킬 표시"를 선택합니다.
![](https://imagedelivery.net/5m26Aj-CutMXPPNacMs_yQ/4e08527e-5768-450d-a0fc-993f10b7a800/public)
11. 메신저에 표시할 스킬 유형을 선택합니다.
![](https://imagedelivery.net/5m26Aj-CutMXPPNacMs_yQ/b4efde1a-d9d2-443c-b401-bde597e3e100/public)
12. 이후, "스킬 옵션 추가"를 클릭하고 모든 스킬을 입력합니다.
![](https://imagedelivery.net/5m26Aj-CutMXPPNacMs_yQ/80498b76-848b-494d-003b-3663e66d9c00/public)
13. "세일즈 팀에 문의" 제목을 복사합니다.
14. 다음 값이 생성된 스킬에 적용됩니다.
- 스킬 선택 - 생성된 스킬 선택 및 입력
- 라벨 작성 - 메신저에 표시할 단어를 작성하며, 사용자가 이해하고 클릭할 수 있는 단어여야 합니다.
- 메시지 응답 - 스킬을 선택한 후 표시할 단어를 입력합니다.
![](https://imagedelivery.net/5m26Aj-CutMXPPNacMs_yQ/1c8e498b-a14a-423c-c49d-81e5e2ba5c00/public)
15. 위의 3가지를 모든 생성된 스킬에 대해 수행합니다.
![](https://imagedelivery.net/5m26Aj-CutMXPPNacMs_yQ/cd9dc19a-e90d-43d7-e007-4c10b99b2300/public)
16. 완료되었다고 생각되면 저장 버튼을 클릭합니다.
![](https://imagedelivery.net/5m26Aj-CutMXPPNacMs_yQ/376f3b55-0929-43b4-744d-731281799f00/public)

# 스킬 테스트 방법
1. 메신저에서 < >를 선택하고 클릭합니다.
![](https://imagedelivery.net/5m26Aj-CutMXPPNacMs_yQ/ad909317-8f05-4d53-5ed2-c36cbf1e9800/public)
2. 이후, 시뮬레이션 버튼을 클릭합니다.
![](https://imagedelivery.net/5m26Aj-CutMXPPNacMs_yQ/93b5b55e-afd1-4d8b-2dab-a398c2f30b00/public)
3. 생성된 스킬이 표시됩니다. 적합한 것을 선택하세요.
![](https://imagedelivery.net/5m26Aj-CutMXPPNacMs_yQ/059d19f8-b20e-427f-b192-6f18ae6d2100/public)
4. 생성한 응답이 동일하게 표시됩니다.
![](https://imagedelivery.net/5m26Aj-CutMXPPNacMs_yQ/93acfd15-5722-4431-4c12-cdc4a7e6ce00/public)
